# Dashboard & Widgets

The dashboard displays configurable widgets that visualize your expense data.

## Available widgets

| Widget | Description | Size |
|---|---|---|
| **Total Stats** | Summary cards: total spent, transaction count, category count | Full width |
| **Spending by Category** | Bar chart breakdown of spending per category | Half width |
| **Biggest Expense** | Shows the single largest expense this month | Half width |
| **Monthly Trend** | Total spending per month as a bar chart | Half width |
| **Most Frequent** | Top 5 most frequently occurring expense titles | Half width |

## Customizing the dashboard

- Click **+ Add Widget** to see available widgets not currently on the dashboard
- Use the **arrow buttons** (← →) above each widget to reorder them
- Click **×** to remove a widget from the dashboard

Your widget configuration (which widgets are shown and their order) is saved automatically and persists across sessions.
