# Category Management

The **Categories** page lets you view, create, rename, merge, and delete your expense categories.

Go to **Categories** in the sidebar to access it.

## Category list

The page shows a table of all categories with:

- **Name** — the category name (click to rename inline)
- **Expenses** — how many expenses use this category
- **Rules** — how many classification rules map to this category

Click column headers to sort by name, expense count, or rule count. Use the search bar to filter categories by name.

## Creating categories

Click **+ New** to create a category. Categories are normally created automatically when you categorize an expense, but you can also create them ahead of time. Duplicate names are blocked.

## Renaming

Click a category name in the table to edit it inline. Press Enter to save, Escape to cancel. Renaming updates the category name across all expenses and classification rules in a single transaction.

## Deleting

Click the **x** button on a category row. A modal asks you to select a replacement category — all expenses and rules from the deleted category will be moved to the replacement. You cannot delete a category without reassigning its data.

## Merging

Select two or more categories using the checkboxes, then click the **Merge** button that appears. Enter a target name (defaults to the first selected category, but you can type any name). All expenses and rules from the selected categories will be consolidated under the target name.
